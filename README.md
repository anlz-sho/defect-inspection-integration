# 不良目視結果統合ツール

## 概要
このツールは、複数のエクセルファイルに分かれて保存されている不良目視結果を一つのエクセルファイルに統合するプログラムです。  
目視画像などの重いデータを除外し、軽量化されたエクセルファイルを生成します。  
これにより、大容量のエクセルを開く際の待ち時間を大幅に短縮でき、目視結果の集計を迅速に行うことができます。

---

## 機能
- **複数エクセルファイルの統合**  
指定したフォルダ内のエクセルファイルを読み込み、一つのエクセルファイルに統合します。

- **不要な画像データの削除**  
ファイル内の目視画像データを削除し、ファイルサイズを軽量化します。

- **高速処理**  
手作業や VBA では時間がかかる処理を、Python を用いて高速化します。

## 使い方
1. **対象フォルダの準備**  
不良目視結果が保存されているエクセルファイルを一つのフォルダにまとめます。
2. **パス設定**  
プログラム内の以下の変数を設定します。
```python
path = "目視結果エクセルを置いたアドレス"
xpath = "目視結果をまとめるエクセルを置いたアドレス"
output = "目視結果をまとめるエクセルのファイル名"
```
3. **プログラムの実行**  
pycharmにてプログラムを実行します。
4. **結果の確認**  
出力先に生成された軽量化されたエクセルファイルを開き、統合結果を確認します。
